# NexaFlow AI — Vercel Deployment Guide

## Files in this folder
- `index.html` → Full website (GEO + SEO optimized)
- `vercel.json` → Vercel configuration

---

## Deploy in 3 Steps

### Step 1 — Upload to GitHub
1. Go to github.com → New Repository
2. Name it: `nexaflow-ai`
3. Drag & drop both files (`index.html` + `vercel.json`) into the repo
4. Click "Commit changes"

### Step 2 — Connect to Vercel
1. Go to vercel.com → Sign up with GitHub
2. Click "Add New Project"
3. Select your `nexaflow-ai` repository
4. Click "Deploy" (no settings needed)

### Step 3 — Custom Domain (Optional)
1. In Vercel dashboard → Settings → Domains
2. Add your domain (e.g., nexaflow.in)
3. Follow DNS setup instructions
